Reason for Hexo + Gitpage

The reason why I want to create a blog is that I want to record what I have learned, which can not only consolidate my memory, but also can share it with you who needs it.

After searching different blog platforms on the Internet and reading everyone’s recommendations, I finally decided to build a blog of my own. No need to pay, no limited, but also I can learn from it, which is a amazing learning process.

what you need

  • Github page: a static web hosting services provided by Github, you can use it to depoly your static webpage of blog or project. And it support Hexo static web frame.
  • Hexo: a static web frame
  • Next: The theme for blog
  • Typora: a markdown editor to help you write blog in Markdown language
